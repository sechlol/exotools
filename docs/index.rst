About exotools
===============

Tools for working with exoplanet data. This package provides utilities for accessing and analyzing exoplanet datasets.

Main Features
-------------

* Access to known exoplanet catalogs
* Tools for working with candidate exoplanets
* Integration with TESS data
* Lightcurve analysis tools
* Database utilities for astronomical data

Installation
-----------

.. code-block:: bash

   pip install exotools

For development installation:

.. code-block:: bash

   pip install -e ".[dev]"
